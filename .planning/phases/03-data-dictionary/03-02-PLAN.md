- Return type: `Promise<{ data: DictionaryDomain[]; error?: undefined } | { data?: undefined; error: string }>`
- Query: `supabase.from("dictionary_domains").select("*").order("display_order", { ascending: true })`

`createDictionaryDomain(input: DictionaryDomainInput)`:
- Auth check + tenant lookup (same pattern as createOntologyClass)
- Get next display_order: query max display_order for tenant, add 1 (default 0 if no domains)
- Insert: `{ tenant_id, name: input.name, description: input.description ?? null, domain_area: input.domainArea ?? null, owner: input.owner ?? null, display_order: nextOrder }`
- Handle 23505 duplicate: `"A domain with this name already exists. Choose a different name."`
- `revalidatePath("/dictionary")`

`updateDictionaryDomain(id: string, input: DictionaryDomainInput)`:
- Auth check (no tenant lookup needed — RLS handles update scoping)
- Update: `{ name: input.name, description: input.description ?? null, domain_area: input.domainArea ?? null, owner: input.owner ?? null }`
- `.eq("id", id).select().single()`
- Handle 23505 duplicate
- `revalidatePath("/dictionary")`

`deleteDictionaryDomain(id: string)`:
- Auth check
- Before delete: count fields assigned to this domain via `supabase.from("dictionary_field_domains").select("*", { count: "exact", head: true }).eq("domain_id", id)`
- Delete domain: `supabase.from("dictionary_domains").delete().eq("id", id)` — cascade on `dictionary_field_domains` unlinks fields automatically
- `revalidatePath("/dictionary")`
- Return `{ data: { unlinkedFields: count ?? 0 } }`

`reorderDomains(orderedIds: { id: string; display_order: number }[])`:
- Auth check + tenant lookup
- For each item in orderedIds, run: `supabase.from("dictionary_domains").update({ display_order: item.display_order }).eq("id", item.id)`
- Use Promise.all for the batch update
- `revalidatePath("/dictionary")`

`getFieldCountForDomain(domainId: string)`:
- Query: `supabase.from("dictionary_field_domains").select("*", { count: "exact", head: true }).eq("domain_id", domainId)`
- Return count

- Handle 23505 duplicate: `"A field with this name already exists. Choose a different name."`
- `revalidatePath("/dictionary")`

`updateDictionaryField(id: string, input: DictionaryFieldInput)`:
- Auth check + tenant lookup
- Update dictionary_fields: `.update({ field_name: input.fieldName, field_definition: input.fieldDefinition ?? null, value_type: input.valueType, tagging_method: input.taggingMethod, ai_instruction: input.aiInstruction ?? null, controlled: input.controlled ?? false }).eq("id", id)`
